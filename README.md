# HappyLearning
HelloWorld is the best program I've ever seen.

```c#
// helloworld.cs
using System;

namespace HelloWorldApplication
{
	class HelloWorld
	{
		static void Main(string[] args)
		{
			Console.WriteLine("Hello World");
		}
	}
}
```

```
$ mcs helloworld.cs
$ ./helloworld.exe
Hello World
```
